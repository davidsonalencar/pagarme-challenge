validator-server
directive in module validatorServer

Description

Diretiva used to automate the display of validation messages based on data obtained by the API.

Dependencies

  • VALIDATOR_SERVER_EVENT
  • $compile

Usage

as attribute
<ANY validator-server>
   ...
</ANY>

Example

<form name="formField">
   <div>
      <input type="text" name="field" ng-model="field" validator-server="" />
   </div>
</form>

Example after processing

<form name="formField">
   <div>
      <input type="text" name="field" ng-model="field" validator-server="" />
      <div ng-messages="formField.field.$error" class="uk-form-help-block">
         <div ng-message="ng-message">
            {{formField.field.$error.server}}
         </div>
      </div>
   </div>
</form>
